# ArtExplain

This repository contains code and links to additional findings for the paper submission:

Towards Explainable Image Style Transfer: Metrics for Balancing Style and Content

The paper makes use of Adain, which can be found [here]([(https://github.com/xunhuang1995/AdaIN-style)])

The dataset is taken from [here]([(https://github.com/victorkitov/style-transfer-dataset)]).

Some of the best examples of the metrics used can be found [here]([(https://drive.google.com/drive/folders/1htSPKXbnEFqjsDBOZpbgx-fYhYG0_hef?usp=drive_link)]).

We made use of an alpha of 0.9, but the variation with alpha can be seen [here]([(https://drive.google.com/drive/folders/1t0HSRayVpVuqi3JlFSX9PEviMxUaMFjf?usp=drive_link)]).

All the heatmaps we generated for patch-size 32 are [here]([(https://drive.google.com/drive/folders/1FOTbQSR6rK1Mo8lsSIE_xinSw92Hzxmv?usp=drive_link)]), patch-size 64 are [here]([(https://drive.google.com/drive/folders/1ZgBhqxAm62-KfDQ15n5mpmRP6e68GNVX?usp=drive_link)]) and patch-size 128 are [here]([(https://drive.google.com/drive/folders/1PqjSI5iaUxdWRSIqYgFBtGo2wDya97yr?usp=drive_link)])
